# Photo Preparation Guide - Step by Step

## Quick Steps to Prepare Your Photo

### Option 1: Online Tools (Easiest - No Software Needed)

1. **Crop to Square & Remove Background**:
   - Go to: https://www.remove.bg (removes background automatically)
   - Upload your photo
   - Download the result (transparent background)
   - Then go to: https://www.iloveimg.com/crop-image
   - Crop to square (1:1 ratio)
   - Download as JPG

2. **Or Use Canva** (Free):
   - Go to: https://www.canva.com
   - Create custom size: 600x600 pixels
   - Upload your photo
   - Crop to fit
   - Remove background if needed
   - Download as PNG or JPG

### Option 2: Windows Built-in Tools

1. **Using Paint 3D** (Windows 10/11):
   - Open your photo in Paint 3D
   - Click "Magic Select" to remove background
   - Crop to square (600x600)
   - Save as JPG

2. **Using Photos App**:
   - Open photo in Photos app
   - Click "Edit" → "Crop"
   - Select "Square" aspect ratio
   - Adjust and save

### Option 3: Online Photo Editor (Recommended)

**Photopea** (Free, works like Photoshop):
1. Go to: https://www.photopea.com
2. File → Open → Upload your photo
3. **Crop to Square**:
   - Select "Crop Tool" (C key)
   - Set aspect ratio to 1:1
   - Adjust crop area
   - Press Enter
4. **Remove Background** (if needed):
   - Select → Subject (auto-selects person)
   - Select → Inverse
   - Delete (removes background)
   - Or use Magic Wand tool
5. **Resize**:
   - Image → Image Size
   - Set to 600x600 pixels
   - Save as JPG

## Recommended Photo Specifications

- **Size**: 600x600 pixels (square)
- **Format**: JPG (best) or PNG
- **File Size**: Under 500KB
- **Background**: 
  - Option A: Transparent (PNG)
  - Option B: Solid color (white/light gray)
  - Option C: Professional gradient
- **Quality**: High resolution, clear and sharp

## Step-by-Step: Using Remove.bg + Crop

1. **Remove Background**:
   - Visit: https://www.remove.bg
   - Upload your photo
   - Wait for processing
   - Download (PNG with transparent background)

2. **Crop to Square**:
   - Visit: https://www.iloveimg.com/crop-image
   - Upload the PNG from step 1
   - Select "Square" aspect ratio
   - Adjust crop area (center your face)
   - Download

3. **Resize** (if needed):
   - Visit: https://www.iloveimg.com/resize-image
   - Upload cropped photo
   - Set to 600x600 pixels
   - Download

4. **Compress** (to reduce file size):
   - Visit: https://tinypng.com
   - Upload your final photo
   - Download compressed version

## Final Steps

1. **Name your photo**: `profile-photo.jpg`
2. **Save it** in the `portfolio-website` folder
3. **Push to GitHub**:
   ```bash
   cd portfolio-website
   git add profile-photo.jpg
   git commit -m "Add profile photo"
   git push
   ```
